# Phase 2: Core Intelligence

## Goal

Deepen the decision quality of the orchestrator, research, domain, visual, and GTM agents so the system behaves like a coherent multi-agent planner rather than a static template generator.

## Tasks

1. Expand clarification handling to tailor assumptions to the input idea.
2. Improve market research generation with category-aware competitors, TAM, whitespace, and keyword logic.
3. Improve brand naming and domain ranking heuristics.
4. Make visual concepts depend explicitly on research insights and selected brand direction.
5. Tighten GTM outputs so city priority, channel mix, and week-one actions are traceable to prior memory.
6. Add focused tests for logic branches and edge cases.

## Exit Criteria

- core agent outputs vary meaningfully by idea category
- memory compounding remains intact
- no duplicate clarification prompts are emitted
- `npm run build` succeeds
- phase code review is completed and findings are fixed
