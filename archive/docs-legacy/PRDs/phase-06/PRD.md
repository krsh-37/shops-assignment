# Phase 06 PRD

Approved scope boundary: `PRD-008`, `PRD-009`.

## 1. Problem Summary
The system must produce India go-to-market guidance and Shopify launch assets from already established memory. This phase translates brand direction into launch execution assets.

## 2. In-Scope Items
- India GTM Agent writes launch plan output.
- Shopify Agent writes store asset package output.
- Shopify output uses prior visual palette and research keywords.

## 3. Out-of-Scope Items
- Paid media strategy.
- SEO/GEO content.
- Final report synthesis.

## 4. Assumptions
1. Research, Visual, and GTM data are present before Shopify runs.
2. Shopify output is an artifact package, not a live store.
3. The product catalog size is bounded by the approved phase plan.

## 5. Constraints and Risks
- Shopify must demonstrate memory compounding.
- Missing prior context must block the dependent agent.

## 6. Open Questions
- None.

## 7. Success Criteria
- GTM output includes launch cities, channels, reel ideas, influencer brief, and checklist.
- Shopify output uses visual palette values from memory.
- Shopify copy uses research keywords from memory.
